常用命令

  • openclaw --version查看版本
  • openclaw gateway probe检查 Gateway 是否正在运行
  • openclaw gateway run在当前窗口启动 Gateway
  • openclaw dashboard打开本地网页控制台
  • openclaw doctor执行本地诊断

命令行窗口打开后，直接在命令提示符后输入命令即可，不要输入 1、2 这类序号。
