# OpenClaw Launcher v1.0.0

## 中文说明

首个公开版本，包含两种 Windows 发行形态：

- 安装版
- 便携版

本版重点：

- 基于 Electron 的 Windows 启动器
- 面向 OpenClaw 和 ClawX 的共享数据协同
- 支持共享目录发现、迁移和 `.openclaw` 链接维护
- 支持在启动器里配置模型 API
- 同时提供 installer 和 portable 两种发行资产

## English

Initial public release with two Windows distribution formats:

- installer build
- portable build

Highlights:

- Electron-based Windows launcher
- shared-data workflow for OpenClaw and ClawX
- shared-root discovery, migration, and `.openclaw` junction maintenance
- model API configuration inside the launcher UI
- both installer and portable release assets
